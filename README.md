# spring boot基础工程  
## 包含组件
1、spring boot 热部署工具 devTools  
2、前端渲染框架 thymeleaf  
3、mysql数据库链接  
4、mybatis  
5、json处理工具fastjson  
6、数据库操作通用类tk.mybatis

## 本地安装测试地址:
  localhost:8080/test
